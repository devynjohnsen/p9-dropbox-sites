##Dropbox

####How-to

1. Upload to **Public** folder
2. Right click **Copy Public** link
3. Share URL

####Pros

* Free
* Easy
* No third party
* Easy to understand

####Cons

*Free
*Has to be in public folder - security concern
*Too long winded URL
